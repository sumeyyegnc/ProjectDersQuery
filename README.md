# 📚 ProjectDersQuery - Course Management System

A simple **ASP.NET Core MVC** application for managing **Students, Courses, and Instructors** with authentication, search, reporting, and CRUD operations.
-

## 🚀 Technologies

- ASP.NET Core MVC (.NET 8)
- ADO.NET
- Microsoft SQL Server (LocalDB)
- HTML5, CSS3, Bootstrap 5
- JavaScript & jQuery
- DataTables.js
- Cookie Authentication
